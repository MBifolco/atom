# Atom Combat Fighters

Fighter assets used by CLI matches, curriculum training, and population export.

## Directory Structure

```text
fighters/
├── examples/                # Hand-authored baseline fighters
│   ├── boxer.py
│   ├── counter_puncher.py
│   ├── out_fighter.py
│   ├── slugger.py
│   └── swarmer.py
├── test_dummies/            # Curriculum opponents
│   └── atomic/              # Atomic behavior fighters for staged training
├── AIs/                     # Exported trained fighters
└── registry.json            # Fighter registry generated by build_registry.py
```

## Example Fighters

- `boxer.py`: pressure-focused baseline style
- `counter_puncher.py`: reactive punisher style
- `out_fighter.py`: spacing-focused style
- `slugger.py`: high-impact aggression style
- `swarmer.py`: high-tempo pressure style

## Curriculum Test Dummies

The curriculum trainer uses fighters in `fighters/test_dummies/atomic/` across 5
progressive levels. Representative opponents include:

- `stationary_neutral.py`, `stationary_extended.py`, `stationary_defending.py`
- `approach_slow.py`, `flee_always.py`, `circle_left.py`, `circle_right.py`
- `distance_keeper_1m.py`, `stamina_efficient.py`, `charge_on_approach.py`
- `aggressive_stance_switcher.py`, `balanced_stance_switcher.py`, `retreater.py`

## Common Usage

```bash
# Example vs example
python atom_fight.py fighters/examples/boxer.py fighters/examples/slugger.py --html replay.html

# Trained AI vs example
python atom_fight.py fighters/AIs/fighter_0/fighter_0.py fighters/examples/counter_puncher.py --watch

# Example vs curriculum dummy
python atom_fight.py fighters/examples/boxer.py fighters/test_dummies/atomic/stationary_neutral.py --html dummy_test.html
```
