# Montage Legacy Scripts

Legacy montage generators kept for historical reference.

Current supported tooling:

- `scripts/montage/create_html_montage.py`
- `scripts/montage/create_html_montage_from_existing.py`
- `scripts/montage/create_montage.py`
- `scripts/montage/render_replays.py`

The scripts in this folder are superseded and should not be used for new runs.
