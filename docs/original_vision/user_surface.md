
# 🖥️ Atom Combat — User Surface (Component)

## Purpose
The **User Surface** is how people interact with Atom Combat — the entry point for players, trainers, officials, and spectators.

> The interface between human creativity and machine combat.

---

## Responsibilities
- Provide commands to create, upload, and manage fighters.  
- Queue or spectate fights.  
- Display replays, rankings, and stats.  
- Offer CLI and Web UI variants for accessibility.

---

## Modes of Use
| User Type | Actions |
|------------|----------|
| **Player** | Create fighter, start matches, review replays. |
| **Trainer** | Review telemetry, update fighter logic. |
| **Official** | Certify fighters, publish results. |
| **Spectator** | Watch fights and view rankings. |

---

## Output Surfaces
- **CLI Mode:** text-based match control and summaries.  
- **Web Mode:** modern browser app for leagues and replays.  
- **API:** developer access for automation and analytics.

---

**One-liner summary:**  
*The User Surface is the front door of Atom Combat — where fighters are born, matches begin, and fans engage.*
